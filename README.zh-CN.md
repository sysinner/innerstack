# innerstack

[English](./README.md)

**轻量级容器调度与管理引擎，面向私有 PaaS**

它通过极简架构实现工业级稳定性。

- **零外部依赖** — 内置数据存储、网络、DNS 和网关。
- **单二进制，全角色** — Zone Leader、Host Agent 和 Scheduler 集成于一个二进制文件。
- **极致轻量** — 最小资源占用，无多余复杂度。
- **生产级可靠性** — 自愈生命周期、认证 API、自动 Leader 选举。

## 功能特性

- **容器编排** — 智能调度、多副本部署、自愈生命周期管理与自动恢复、精细化资源隔离。
- **内置 VPC 网络** — 零配置 VXLAN 覆盖网络，自动 IP 地址管理与路由传播，Zone 内全互联。
- **服务发现** — 内置 DNS，自动为所有应用实例提供名称解析，随容器调度实时更新。
- **HTTP 网关** — 基于域名的路由，支持自动 TLS（Let's Encrypt）、gzip 压缩、IP 限速与负载均衡。
- **包分发** — 内置包仓库，支持断点续传的分块上传/下载与完整性校验。
- **应用生命周期** — 任务引擎支持启动、关闭、周期性和 Cron 定时任务，支持依赖排序与自动重试；模板渲染实现配置注入。
- **安全** — 密钥签名认证的 Access Key 与权限范围控制；无需外部协调服务的自动确定性 Leader 选举。

## 项目状态

innerstack v2 目前处于**活跃开发**阶段（alpha）。架构和核心功能已稳定，功能正在逐步增加。
